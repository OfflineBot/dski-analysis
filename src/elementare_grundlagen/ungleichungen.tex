\subsubsection{Betrag}
Der Betrag $|a|$ einer Zahl $a$ ist der Abstand dieser Zahl vom Nullpunkt auf der Zahlengerade:

$|a| = a$ oder $-a$
\begin{itemize}
    \item falls $a \geq 0$
    \item falls $a < 0$
\end{itemize}

Beispiel: $|-5| = -(-5) = 5$


\subsubsection{Binomische Formeln}
\begin{itemize}
    \item $(a + b)^2 = a^2 + 2ab + b^2$
    \item $(a - b)^2 = a^2 - 2ab + b^2$
    \item $(a + b) \cdot (a - b) = a^2 - b^2$
\end{itemize}
